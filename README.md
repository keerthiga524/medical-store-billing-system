# Medical Store Billing System

A simple Python project to manage a medical store. This system helps to add medicines, view available stock, and generate bills when medicines are sold.

## Features

- Add new medicines with price and quantity  
- View available medicines and stock details  
- Sell medicines and generate bill  
- Automatic stock update after selling  
- Menu-driven program for easy use  

## Technologies Used

- Python

## Menu Options

1. Add Medicine  
2. View Medicines  
3. Sell Medicine  
4. Exit
